## 🧩 What this app does

TicTac — Vibe-Coded Isometric Sprites creates 8-way isometric character sprites for tactics games. It helps you make clean character art for top-down and angled game views without drawing each frame by hand.

Use it to:

- Generate isometric character sprites
- Make 8-direction character sets
- Build art for tactics games and RPGs
- Create matching sprite sheets for game engines
- Save time on repetitive sprite work

The app is aimed at end users who want to make game art on a Windows PC and run it with a simple download.
